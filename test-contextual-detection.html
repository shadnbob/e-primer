Context-Aware Detection Test Cases

Test 1: "It seems" + Evidence (Should be EXCELLENCE)

It seems the data supports this hypothesis with strong statistical significance.

Expected: Excellence highlight for nuanced language
Test 2: "It seems" + False Certainty (Should be WEASEL)

It seems obvious that climate change is a hoax without any evidence.

Expected: Weasel highlight for false certainty
Test 3: "It seems to me" + Reasoning (Should be EXCELLENCE)

It seems to me, based on the evidence, that this approach has merit.

Expected: Excellence highlight for transparent opinion
Test 4: "It seems" + Vague Attribution (Should be WEASEL)

It seems like everyone agrees this is the best solution.

Expected: Weasel highlight for vague attribution
Test 5: "Studies show" + Citation (Should be EXCELLENCE)

Recent studies show that exercise improves mental health (Johnson et al. 2023).

Expected: Excellence highlight for specific attribution
Test 6: "Studies show" + No Citation (Should be WEASEL)

Studies show that this treatment is completely safe and effective.

Expected: Weasel highlight for vague attribution
Test 7: "Appears" + Evidence Basis (Should be EXCELLENCE)

This approach appears promising based on preliminary results from the trial.

Expected: Excellence highlight for qualified observation
Test 8: "Appears" + False Certainty (Should be WEASEL)

It appears obvious that this policy will fail completely.

Expected: Weasel highlight for false certainty
Test 9: Mixed Context - Complex Sentence

According to Dr. Smith, it seems the research indicates positive outcomes, though experts believe more studies are needed.

Expected: Excellence for "it seems" with attribution, weasel for "experts believe"
Test 10: Neutral Case

It seems reasonable to consider this option among others.

Expected: No highlighting (neutral usage overrides regular weasel detection)

What to Look For

Enhanced Contextual Tooltips: Phrases like "it seems" will now show different explanations based on context:

  • Hover tooltips: Show specific reasoning with confidence levels
  • Click tooltips: Show detailed "Context Analysis" section with full reasoning
  • Different classifications: Same phrase can be excellence, weasel, or neutral based on surrounding words

🎯 Key Enhancement: The tooltip now shows the full analyzed context window (up to 100 characters before/after the phrase) with the matched phrase highlighted in yellow. This helps you understand exactly what text the AI analyzed to reach its conclusion.
🔄 Neutral Override: When the context analysis determines a phrase is neutral (appropriate usage), it overrides the regular bias detection and shows no highlighting. This prevents false positives where contextually appropriate language would otherwise be flagged.

Testing Instructions

  1. Build the extension: npm run build
  2. Load the extension in Chrome developer mode
  3. IMPORTANT: If you had the extension loaded before, click the "🔄" reload button in chrome://extensions to ensure the new code is loaded
  4. Navigate to this test file
  5. Check the highlighting patterns against expected results
  6. Hover over highlights to see brief contextual reasoning
  7. Click on highlights to see detailed context analysis

Troubleshooting

⚠️ If neutral case still shows as weasel:
  1. Reload the extension - Click the reload button (🔄) in chrome://extensions
  2. Check settings - Make sure "Weasel Words" is enabled in the popup
  3. Check mode - Make sure analysis mode is "Balanced" or "Problems", not "Excellence only"
  4. Hard refresh - Reload this page with Ctrl+F5 or Cmd+R
  5. Debug console - Open DevTools (F12) and look for console logs starting with "[BiasDetector]"
🔍 Debug Mode Active: The extension now logs detailed information to the browser console. For Test 10 ("It seems reasonable..."), you should see:
