# Bias Detector Info Page

## Overview
Created a comprehensive user guide that explains:
- What each bias category detects
- Why each type of bias matters
- Examples of biased vs. neutral language
- Important limitations and context considerations
- Tips for effective use

## Features

### Clear Explanations
Each bias type includes:
- **Description**: What patterns it detects
- **Examples**: Specific words and phrases
- **Why it matters**: The impact on readers
- **Context notes**: When it's appropriate vs. problematic

### Visual Design
- Color-coded sections matching the extension
- Clean, readable layout
- Highlighted example text
- Clear warning boxes for limitations

### Educational Content
- How to use the tool effectively
- Rewriting exercises
- Genre-specific guidance
- Cultural considerations

## Access
Click "Learn about bias detection categories »" at the bottom of the extension popup

## Purpose
Helps users:
1. Understand what the highlights mean
2. Learn why these patterns matter
3. Recognize the tool's limitations
4. Become more critical readers
5. Write more clearly and fairly

The guide emphasizes that context matters and that the tool identifies *potential* bias indicators, not definitive bias.
